# Flyway CLI Video Series

A comprehensive video series to help users get started with the Flyway CLI.

## Series Overview

| Video # | Topic | Duration (Est.) | Difficulty |
|---------|-------|-----------------|------------|
| 1 | Project Initialization with `flyway init` | 5-8 min | Beginner |
| 2 | Comparing Changes with `flyway diff` | 5-7 min | Beginner |
| 3 | Visualizing Changes with `flyway diffText` | 3-5 min | Beginner |
| 4 | Building Your Schema Model with `flyway model` | 5-7 min | Beginner |
| 5 | Generating Migrations with `flyway generate` | 7-10 min | Intermediate |
| 6 | Applying Migrations with `flyway migrate` | 5-7 min | Beginner |
| 7 | Viewing Migration Status with `flyway info` | 3-5 min | Beginner |
| 8 | Validating Migrations with `flyway validate` | 3-5 min | Beginner |
| 9 | Preparing Deployment Scripts with `flyway prepare` | 5-7 min | Intermediate |
| 10 | Deploying with `flyway deploy` | 5-7 min | Intermediate |
| 11 | Managing Database Cleanup with `flyway clean` | 5-7 min | Intermediate |
| 12 | Creating Snapshots with `flyway snapshot` | 5-7 min | Advanced |
| 13 | Baselining Existing Databases with `flyway baseline` | 7-10 min | Advanced |
| 14 | Undoing Migrations with `flyway undo` | 5-7 min | Advanced |
| 15 | Testing Connections with `flyway testConnection` | 2-3 min | Beginner |

## Prerequisites

- Flyway Enterprise Edition installed and licensed
- SQL Server instance available
- PowerShell 5.1+ or Terminal access
- Basic SQL knowledge
