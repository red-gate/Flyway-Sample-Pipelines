# Flyway CLI Video Series

A comprehensive video series to help users get started with the Flyway CLI.

> **Note on database engines:** The examples in this series use SQL Server, but these commands work the same way for every database where Flyway has Advanced capabilities - including Oracle, PostgreSQL, and MySQL. Flyway also supports the cloud-hosted versions of these platforms (e.g. Azure SQL, Amazon RDS, Aurora, Google Cloud SQL). See the full list of [supported databases and versions](https://documentation.red-gate.com/fd/supported-databases-and-versions-143754067.html).

## Series Overview

| Video # | Topic | Duration (Est.) | Difficulty |
|---------|-------|-----------------|------------|
| 1 | Project Initialization with `flyway init` | 5-8 min | Beginner |
| 2 | Comparing Changes with `flyway diff` | 5-7 min | Beginner |
| 3 | Visualizing Changes with `flyway diffText` | 3-5 min | Beginner |
| 4 | Building Your Schema Model with `flyway model` | 5-7 min | Beginner |
| 5 | Generating Migrations with `flyway generate` | 9-12 min | Intermediate |

## Prerequisites

- Flyway Enterprise Edition installed and licensed
- SQL Server instance available
- PowerShell 5.1+ or Terminal access
- Basic SQL knowledge
